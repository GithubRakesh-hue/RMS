# 🍕 Restaurant Management System (Python)

A simple command-line based Restaurant Management System built using Python. This project allows users to manage menu items, place customer orders, remove items from orders, view order summaries, and generate bills.

## 📌 Features

* View restaurant menu
* Add new items to the menu
* Place customer orders
* Remove items from an order
* Display order summary
* Generate final bill
* Simple menu-driven interface

## 🛠 Technologies Used

* Python 3
* Dictionaries
* Functions
* Loops and Conditional Statements
